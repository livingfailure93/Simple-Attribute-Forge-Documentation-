Simple Forge Product Family

Simple Attribute Forge (Free)

Core attribute system with wizard, code generation, regeneration, and events. Free on the Unity Asset Store.

Influence System

Formula-driven attribute calculations, 7-step pipeline, and breakpoints. Included in Ultimate.

Simple Attribute Forge Ultimate

Complete package with Modifier System and Character Templates. Everything you need for professional RPGs.

Upgrade Path: Each tier builds on the previous one. Base users can upgrade to Pro or Ultimate without losing any existing work.
