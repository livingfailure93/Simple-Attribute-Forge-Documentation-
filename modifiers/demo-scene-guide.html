Demo Scene Guide

Walkthrough of the included SimpleAttributeForgeUltimate demo scene.

Opening the Demo

  1. Navigate to: Assets/Simple Attribute Forge Ultimate/Scenes/
  2. Open SimpleAttributeForgeUltimate.unity
  3. Press Play

Demo Features

Character Template Selection

Use the dropdown to select different character classes. The starting attributes will update based on the selected template.

Modifier Application

Use the modifier dropdown to select and apply different modifiers. See how they affect attribute values in real-time.

Attribute Display

The UI shows all attributes with their current values, including:

  • Base values
  • Formula bonuses (from Influence System)
  • Modifier bonuses (from active modifiers)
  • Total calculated values

Active Modifier List

View all currently active modifiers with their remaining durations and stack counts.

Demo Components

SimpleAttributeUltimateDemoManager

Main demo script that handles:

  • Template dropdown population
  • Modifier dropdown population
  • Attribute value display updates
  • Modifier application/removal

Generated Assets Used

  • SoulslikeCharacterAttributeData
  • SoulslikeInfluenceSystem
  • 12 modifier assets (Adrenaline Surge, Frostbite, etc.)
  • 13 character template assets

Experimenting

Try these experiments to understand the system:

  1. Apply multiple modifiers and see how they stack
  2. Try conditional modifiers (Adrenaline Surge) - lower health to see effects
  3. Switch between character templates and observe starting stat differences
  4. Check how Influence formulas calculate derived stats from base attributes
