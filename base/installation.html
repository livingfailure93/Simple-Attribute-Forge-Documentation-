Installation

Unity Asset Store

  1. Open Unity Asset Store in your browser or Package Manager
  2. Search for "Simple Attribute Forge"
  3. Download and import the package
  4. Wait for Unity to compile the scripts

Unity Package Manager

  1. Open Unity Package Manager (Window → Package Manager)
  2. Click "My Assets" dropdown
  3. Find "Simple Attribute Forge" and click Import
  4. Wait for installation to complete

Manual Installation

  1. Download the Simple Attribute Forge .unitypackage file
  2. Double-click the .unitypackage file to import
  3. Check all files and click Import
  4. Wait for Unity to compile the scripts
Installation Complete!
Access the wizard via: Window → Simple Attribute Forge → Attribute Data Wizard

Requirements

  • Unity 2021.3 or newer
  • No external dependencies required
