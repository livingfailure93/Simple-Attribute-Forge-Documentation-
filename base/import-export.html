Import/Export Checklist

Before Exporting

  • All attribute names are meaningful
  • Descriptions are complete and helpful
  • Behavior types are correctly assigned
  • Values and ranges make sense for your game
  • Icons are assigned where needed

Before Importing

  • JSON file is from a trusted source
  • Backup your current configuration
  • Understand the import strategy (Replace vs Add)
  • Review the JSON structure for validity
  • Check for potential naming conflicts

After Import

  • Review all imported attributes
  • Check validation messages
  • Verify behavior types are correct
  • Test the wizard workflow end-to-end
  • Generate code to verify everything works
